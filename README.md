# Pauli REPL

Small Rust CLI/REPL for symbolic simplification of 2x2 Pauli algebra.

## Quickstart

```bash
cargo run
```

Demo inputs:

- `X*Y`
- `[X,Y]`
- `(X+Y)*(X+Y)`

Built-in commands:

- `help`
- `examples`
- `exit`
